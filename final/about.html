About Lagos Heritage Hub

Lagos Heritage Hub is a student project showcasing cultural and historical places across Lagos. The site demonstrates responsive design, accessible components, client-side data handling with JavaScript ES Modules, localStorage, and dynamic DOM manipulation.

Attributions

Images, historical data and copy used on this site should be properly attributed. Example resources:

  • Public domain images and local photographers (add details here)
  • Wikipedia for historical references
  • Local archives and oral history sources

Project Plan

This project uses a local data file data/attractions.json, separate CSS and JS, and follows semantic HTML and accessibility best practices.
